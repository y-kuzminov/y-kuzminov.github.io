# API Testing — Postman

**Object:** reqres.in (public REST API).
**Covered:** GET / POST / PUT / DELETE, status codes, response body & fields, basic schema.

## How to run
1. Import `collection.json` into Postman.
2. Run the collection (Runner) — all tests should pass.
3. CLI option: `newman run collection.json`

## What it shows
Request building, assertions in Postman tests, positive & negative checks (e.g. 404).
